Please look at the 1.9.0 changes, this is just a small bugfix update.

### Changes
- Updated to Complementary Shaders r5.8.1 [YOUTUBE SHOWCASE](https://www.youtube.com/watch?v=onb7BZcHu1Y)
- Added Reimagined Clouds outline thanks to TerexG
- Allow higher than 1.0 cloud opacity to make them more opaque
- Don't reduce vanilla lightmap on light sources themselves with photonics, improving their look significantly
- No longer force unbound clouds when pixelated shadows are enabled
- Added a simple tonemap literally just pow(color, vec3(1.0 / 2.2));
- Added a way to disable the old photonics error message
- Added more modded blocks to the properties files thanks to the contributors <3
- Updated translations thanks to the contributors <3

### Fixes

- Fixed the reimagined sun still showing if unbound one is used when a skybox mod is installed
- Fixed some more voxy compile issues
- Fixed other compile errors
- Fixed create aeronautics nameplate rendering incorrectly with generated normals
- Fixed disabling photonics blocklight not working
- Fixed Modded End Biome Tint setting not working correctly
