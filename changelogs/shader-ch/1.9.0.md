### Changes

- Updated to Complementary Shaders r5.8 [YOUTUBE SHOWCASE](https://www.youtube.com/watch?v=onb7BZcHu1Y)
- Added Photonics support! Enabling real-time blocklight shadows and global illumination! Requires Photonics 0.3+
- Added a new Pixelated Unbound Clouds setting which pixelates the clouds
- Improved custom skybox support significantly! This includes skyboxes from mods and resource packs!
- Removed the End Beams Near Player option but added: Beam scale, beam distance, beam height, beam power and more!
- Added a vertical End beam direction bias setting
- Added End beam End flash customization
- Separate the Euphoria Patches Popular Settings profile from the performance profile thanks to an exclusive feature from the mod! On Angelica for 1.7.10 this functionality has been added directly by me. The new location for the popular settings profile is right in the Euphoria Patches Settings menu.

- Added a manual mode for pixelated shadows to combat mismatched RP resolutions
- Fixed no rain above clouds being garbage :D
- Removed `ACT_DISTANCE_WARNING_OVERRIDE` hidden setting, as setting `COORDINATES_ACT_ERROR=false` works just fine
- Added a warning for when 3d skin layers + player ws reflections enabled
- Added a sun/moon ipbr option
- Improved DH leaf color
- Added a Cloud Shadow Distance setting, to remove cloud shadows near the player
- Added a Cloud Shadow Strength setting
- Pixelated Nether Portal noise is now an option
- Added rain opacity and snow opacity multiplier settings
- Remove darkness desaturation when aurora influence is triggered.
- Add a new option to remove darkness desaturation from the sky
- Improved handheld directional lightmap normals accuracy thanks to plazmal
- Removed purple end fire from lava when not using IPBR to be consistent
- End flash shadows now change the normals - how was it always this broken??
- End flash shadows intensity can now be adjusted
- Added an end beam speed setting!
- Added an end star hemisphere option
- Balance the end flash shadows colors in such a way so that the shadows equal the ambient lighting color when there is no flash
- Added End beams Terrain Blend setting
- Slightly tint the illuminated area when an end flash happens with end flash shadows enabled with the flash color
- Added the most horrendous code to fix ender dragon death beam shadows still being visible during the death animation with Euphoria Patches dragon death effect enabled
- No longer cast dragon healing beam shadows when using EP volumetric healing beam
- Made disabling Remove Near Lights for darkness desaturation also remove the ACT light fog one
- Added a Hide Vanilla Stars option
- Allow higher values of cloud light/ambient multipliers
